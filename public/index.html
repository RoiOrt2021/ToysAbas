Toys Abas

How to manage routes and all API-related requests

  • /toys  :  GET - You will see all the toys available in the API.
  • /toys/search/?s=value  :  GET - Search for specific toys by search value (replace 'value' with name/letter etc..).
  • /toys/category/catName  :  GET - Search for specific toys by category (replace 'catName' with the category name).
  • /toys/prices/?min=x&max=x  :  GET - Search for toys by price range. (replace 'x' with the price value.)
  • /toys/ :  POST - Add a toy to the API. Must add a name, info, category and price properties.
  • /toys/editId  :  PUT - edit a toy property by ID (replace 'editId' with the actual toy ID)
  • /toys/delId  :  DELETE - delete a toy by ID (replace 'delId' with the actual toy ID)
